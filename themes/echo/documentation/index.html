1 . Getting Started

For the purposes of this documentation, it is assumed that you have Ghost up and running. If not, please visit support.ghost.org and follow the guide to install and configure Ghost.

2. Installation

  1. Navigate to the folder on your server where Ghost is installed.
  2. Once there, navigate to the /content/themes/ folder.
  3. Upload the theme files to this folder.
  4. Restart Ghost.
  5. Log into your Ghost Admin.
  6. Navigate to Settings > General and select Echo in the theme options dropdown menu.
  7. Click 'Save'.

Note: If you're on using Ghost(Pro), please follow these instructions on how to install a theme.

3. Upgrading

If you have made any changes to Echo's template or SCSS/CSS files, you will need to re-integrate your custom theme changes.

Always perform a backup of the theme files before upgrading

  1. Download the latest version of Echo from your purchases page on Creative Market.
  2. Navigate to the folder on your server where Ghost is installed.
  3. Once there, navigate to the /content/themes/ folder.
  4. Upload and replace all of the older theme files with the newer ones.

Once you update your theme, you will want to restore your changes to the custom.css & theme-options.js files.

4. Settings

In Settings > General several options will alter the content in Echo:

  • Blog Logo - Appears as the first item the header. We recommend a maximum size of 250 Ã— 45 pixels.
  • Blog Title: - The blog title will be displayed if no blog logo has been uploaded.
  • Blog Description - Appears as the first item in the sidebar.
  • Blog Cover - This will appear as a full background image in the underneath the header.
  • Posts per page - This is how many posts are displayed per page.

5. Customization

If you're comfortable with HTML & CSS, you can make any changes to the theme by editing Echo's template files.

Please be aware that when Echo is updated in the future, you will need to re-integrate your custom theme changes.

The CSS files were created using SASS. If you are familiar with SASS, you can easily customize Echo.

Using CSS

You can add your own custom CSS to the custom.css stylesheet. This file is located in the assets > css folder.

Using SASS

You will find the following SCSS files in /assets/scss/ folder:

  • screen.scss - This is the main CSS stylesheet. It imports the mixins.scss and variables.scss files.
  • mixins.scss - This contains all the CSS declarations that Echo reuses throughout the theme.
  • variables.scss - This contains all the variables that you can tweak to customize the way Echo looks.

Note: You will need a SASS preprocessor, such as Grunt to use the SCSS files.

Navigation

You can customize the navigation menu by editing the navigation.hbs file located in the partials folder.

This is a great place to put links to your static pages or custom links to your website.

  1. Navigate to the partials folder on your server.
  2. Open the navigation.hbs file in your favorite text editor.
  3. Replace the hashtags (#) with your own links.
  4. Upload the changed navigation.hbs file to the partials folder.

Example With Absolute Links:

<li>
    <a href="http://www.your-website-here.com">About Me</a>
</li>
<li>
    <a href="http://www.your-website-here.com">Resume</a>
</li>

Example With Static Page Links:

<li>
    <a href="about">About Me</a>
</li>
<li>
    <a href="resume">Resume</a>
</li>

Social Media

Add your own social media links to the header by editing the theme-options.hbs file located in the partials > js folder.

  1. Navigate to the partials > js folder on your server.
  2. Open the theme-options.js file in your favorite text editor.
  3. Find the Social Media section

  4. Add your social media links (between the quotation marks) to the corresponding social media variables.

    Facebook Example:

     var facebook_link = "http://www.facebook.com/my_facebook_username";

    Email Example:

     var email_link = "mailto:youremail@gmail.com";

  5. Upload the changed theme-options.js file to the partials > js folder.

Note: If you don't want a specific social media icon to appear, just leave variable blank.

var facebook_link = "";

Note: If you wish to show RSS icon, change the rss_link value to true.

var rss_link = true;

Cover Photo Text

You can add custom text to appear on top of the cover photo by editing the theme-options.js file located in the partials > js folder.

This is a great place to put your blog's tagline or motto.

  1. Navigate to the partials > js folder on your server.
  2. Open the theme-options.js file in your favorite text editor.

  3. Find the cover_photo_text variable and replace the text (between the quotation marks):

    Example:

     var cover_photo_text = "My super awesome Ghost blog";

  4. Upload the changed theme-options.js file to the partials > js folder.

Note: If you don't want the cover photo text to appear, just leave the cover_photo_text variable blank.

var cover_photo_text = "";

Comments

You can enable Disqus comments on your blog by adding your Disqus shortname to the theme-options.js file located in the assets > js folder:

  1. Add your site to your Disqus account and setup a unique Disqus URL. The unique shortname you pick for your Disqus URL will be important to remember!
  2. Navigate to the assets > js folder on your server.

  3. Open the theme-options.js file and add your unique shortname to the disqus variable.

    Example:

     var disqus = "DISQUS_SHORTNAME_HERE";

  4. Save/update the theme-options.js file.

Note: If you don't want comments on your blog just leave the disqus variable blank.

var disqus = "";

Favicon

You can add you own custom Favicon to your blog by replacing the favicon.ico file located in the assets folder:

  1. Navigate to the assets folder on your server.
  2. Replace the favicon.ico file with your own custom icon.

There are many different web based tools for converting an image into a favicon.ico file.

Post Cover Images

You can add a post cover image to your posts by adding one in the post settings. This post cover image will be displayed along with the title of the post on the homepage.

  1. Login to the Ghost Admin
  2. Create or edit a post.
  3. Click the post settings icon, this is located next to the publish/update post button.
  4. Click "Add post image.
  5. Save/update the post.

Image Lightbox

By default Echo will automatically add an image lightbox to all images within a post. You can disable the automatic image lightbox by editing the theme-options.js file located in the partials > js folder.

  1. Navigate to the partials > js folder on your server.
  2. Open the theme-options.js file in your favorite text editor.

  3. Find the lightbox variable and change the value from true to false.

    Example:

     var lightbox = false;

  4. Upload the changed theme-options.js file to the partials > js folder.

Note: If you wish to enable the automatic image lightbox again just change the false value to true.

var lightbox = true;

Infinite Scroll

By default infinite scroll will be disabled. You can enable infinite scroll by editing the theme-options.js file located in the partials > js folder.

  1. Navigate to the partials > js folder on your server.
  2. Open the theme-options.js file in your favorite text editor.

  3. Find the lightbox variable and insert the text (between the quotation marks) scroll or click. Here are the differences between the two options:

    Scroll will load new posts while scrolling:

     var infinite_scroll = "scroll";

    Click will load new posts with a click of a button:

     var infinite_scroll = "click";

  4. Upload the changed theme-options.js file to the partials > js folder.

Note: If you wish to disable infinite scroll just change the leave the infinite_scroll variable blank.

var infinite_scroll = "";
